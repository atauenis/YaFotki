Данная программа предназначена для вывода графики и данных с Fotki.Yandex.Ru.
По состоянию на текущую версию, доступен вывод:
-Самих фотографий;
-Названий, описаний, меток, временных отметок;
-Открытых альбомов (закрытые не поддерживаются, на время бэкапа снимайте пароль);
-Списка меток или списка всех фото.

Программа находится в разработке, и имеет право глючить. О багах пишите на GitHub.

Системные требования: .NET Framework 4.5 или Mono с поддержкой CLR 4.5, 64 МБ памяти.
Поддержка .NET 4.5 встроена в Windows 8, 8.1, 10 и свежие сборки Linux.
Работа на Linux не тестировалась, пути для сохранения допускаются только относительные.

У программы три режима работы:
1.) Демонстрация данных с сервера (проверка связи). Имитирует веб-интерфейс сайта.
2.) Демонстрация данных из локальной базы (копии данных с сервера)
3.) Резервное копирование данных с сервера в локальную базу.

По умолчанию активен первый режим. Для включения скачивания, необходимо поставить
галочку "Сохранять". Тогда всё, что видит пользователь, будет скачано на диск в
локальную базу. Для массовой выкачки альбомов или фотографий используйте кнопку
с изображением дискеты в окне просмотра альбомов.

Если открывается много окон, это не баг, а фича. Все вопросы к разработчикам
API сервиса Яндекс.Фотки.

Для просмотра локальной базы необходимо ввести путь к папке базы и логин
пользователя, которому принадлежат фото в базе. Затем надо поставить галочку
"Из копии", и нажать кнопку "Загрузить".

Всё вытянутое с Fotki.Yandex.Ru сохраняется в том виде, в котором выдаёт API сервиса.
Просмотреть можно только в данной программе. Экспорт в другие форматы каталогов,
а также на другие сайты запланирован на неопределённый срок.
